Real Time Collaboration System

A web platform for students and professors to manage topics, course tasks, submissions, notifications, and collaborative academic workflows in one place.

View GitHub Repository View Demo Sections

Main Features

Designed around the practical academic collaboration flow between professors and students.

01

Role-based workspace

Separate student and professor experiences for managing dashboard actions.

02

Topic selection

Students can work with available topics while professors organize topic batches.

03

Course tasks

Professors create tasks and review submissions from student groups.

04

Notifications

System notifications keep users informed about important collaboration updates.
